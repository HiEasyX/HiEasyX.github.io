控件布局

如果你想对 HiEasyX 有一个完整的认识，控件布局是你必须要了解的一项东西。什么是控件布局？简单点来说就是：控制控件怎么排列的。

让我们最后一次回看上次的测试代码，你就会发现还有一些东西是我们没见过的。这里我已经给你标出来了：

HX::BeginSameLine();
if (HX::Button(HXStr("Hello Button"), buttonProfile)) {
	++count;
}
HX::Text(HXStr("Count : ") + ToHXString(count));
HX::EndSameLine();

嗯，HX::BeginSameLine 和 HX::EndSameLine 是什么意思？如果试着把这两行代码删掉。你就会发现按钮和文本框出现在了不同行上。而不是在同一水平线上。是的，HiEasyX 中，所有控件都是默认垂直排列的。

如果你想要你的控件水平排列，那么你可以使用 HX::BeginSameLine 打开水平排列功能，这样所有的控件都会出现在同一水平线上了，在你结束水平排列以后，用 HX::EndSameLine 关掉它。但需要注意有部分控件是不支持水平排列的，详情请参见文档。此处不再赘述。
