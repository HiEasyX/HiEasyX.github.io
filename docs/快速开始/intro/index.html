快速上手 HiEasyX

首先，感谢你愿意尝试它 HiEasyX，但在一切开始之前，我们希望你能够先去右上角的 Gitee 仓库主页上给我们的项目点一个免费的 star⭐。这将激励我们继续开发和完善 HiEasyX，谢谢！

本文将会介绍如何配置 HiEasyX 库，完成本文后，你将会在你的开发环境下配置 HiEasyX 并创建一个简单的界面。我们假设你已经有一定的 C 语言基础，并且能够比较熟悉地使用 EasyX 的相关接口。

首先，先将 HiEasyX 从远程代码仓库中拉取下来：

git clone https://gitee.com/gfigure/hi-easyx
提示

如果你不知道这是什么也没关系，你可以直接前往 Gitee 中，通过“发行版”栏目下载源码并解压。

如果你已经准备好了 HiEasyX 的源码的话，那一切都就绪了，让我们开始吧。

确定你的开发环境​

HiEasyX 兼容的最低 EasyX 版本为 2023-7-23（推荐从官网上下载最新版本）。如果你不确定你的 EasyX 版本的话，可以用你的 IDE 创建一个项目，并试试看存不存在 ExMessage 这个结构体，如果存在，则 HiEasyX 兼容你正在使用的 EasyX 版本。

如果是 Visual Studio 用户​

目前为止我们测试了 Visual Studio 2022 和 Visual Studio 2019，两者均可以在默认配置下正常运行 HiEasyX。但这并不代表往前的 Visual Studio 不受支持，理论而言 HiEasyX 的最低支持版本为 Visual Studio 2010，但我们尚未使用相关环境测试。尽管如此，如果你在使用其他 Visual Studio 版本使用 HiEasyX 时遇到问题，我们欢迎你前往 Gitee 的 issue 栏目向我们反馈。

如果你正在使用 Visual Studio，请查看这篇文章来配置 HiEasyX

如果你是 MinGW 用户​

对于 MinGW 用户，请确保你的编译器能够完整支持 C11 标准。

如果你正在使用 MinGW，请查看这篇文章来配置 HiEasyX
